# AGENDA TELEFÔNICA

## ABOUT
Agenda telefônica criada para auxílio na gestão de contatos, na qual está incluso as seguintes funcionalidades: Adicionar, Remover, Editar, Buscar contato. Basicamente realiza um CRUD simples.
